🌤 Weather Application

A modern desktop Weather Application built using Python and Tkinter that fetches real-time weather information from the OpenWeatherMap API. The application provides accurate weather details through a simple and user-friendly graphical interface.



📌 Project Overview

This Weather Application allows users to enter the name of any city and instantly view its current weather conditions. The application communicates with the OpenWeatherMap API, processes the JSON response, and displays the information in a clean GUI.



✨ Features

- 🌍 Search weather by city name
- 🌡 Displays current temperature
- ☁ Shows weather description
- 💧 Displays humidity
- 🌬 Shows wind speed
- 📊 Displays atmospheric pressure
- 🤗 Shows "Feels Like" temperature
- 🕒 Displays current date and time
- ⚠ Error handling for invalid city names
- 🌐 Handles internet connection errors



🛠 Technologies Used

- Python
- Tkinter
- Requests Library
- OpenWeatherMap API
- JSON
- Datetime Module

▶ How to Run

1. Install Python (3.10 or later recommended).
2. Install the required library:

pip install requests

3. Create a free API key from OpenWeatherMap.
4. Replace:

API_KEY = "YOUR_API_KEY"

with your own API key.

5. Run the application:

python "python mini project.py"
